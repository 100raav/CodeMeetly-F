CodeMeetly is a modern live coding interview platform that combines a real‑time collaborative code editor, HD video/audio calls, screen sharing, and chat – all in one beautiful glass‑morphism interface. Designed for seamless technical interviews and pair programming.

🌐 Live Demo: https://code-meetly-f.vercel.app/

✨ Features
Real‑time Collaborative Code Editor (Ace editor, multi‑language)

Video & Voice Calls with colourful mute/unmute buttons

Screen Sharing – replaces your video tile when active (Zoom‑like)

Live Chat – group and private messaging

Role‑based Question Management – interviewer types, interviewee reads

Room Creation & Joining – password‑protected rooms, up to 1000 participants

Google Login (mock) – one‑click authentication

Practice Mode – simulated interview environment

Dark / Light Theme Toggle – smooth transitions

Fully Responsive – works on desktop, tablet, and mobile

Note: The current live demo is a frontend prototype with simulated backend interactions. The full Java backend (Spring Boot + WebSockets) is under development and will be integrated soon.
