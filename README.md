CodeMeetly is a modern live coding interview platform that combines a real‑time collaborative code editor, HD video/audio calls, screen sharing, and chat – all in one beautiful glass‑morphism interface. Designed for seamless technical interviews and pair programming.

🌐 Live Demo: https://code-meetly-f.vercel.app/

✨ Features
1. Real‑time Collaborative Code Editor (Ace editor, multi‑language)
2. Video & Voice Calls with colourful mute/unmute buttons
3. Screen Sharing – replaces your video tile when active (Zoom‑like)
4. Live Chat – group and private messaging
5. Role‑based Question Management – interviewer types, interviewee reads
6. Room Creation & Joining – password‑protected rooms, up to 1000 participants
7. Google Login (mock) – one‑click authentication
8. Practice Mode – simulated interview environment
9. Dark / Light Theme Toggle – smooth transitions
10. Fully Responsive – works on desktop, tablet, and mobile

Note: The current live demo is a frontend prototype with simulated backend interactions. The full Java backend (Spring Boot + WebSockets) is under development and will be integrated soon.
